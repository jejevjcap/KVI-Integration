# API KVI Integration

CAP's OCR API PROJECT FOR TAM

## Pre-requisites

Python 3.11.0 Installed

## Installation

Clone This Repository

Create Python Environment

```bash
  cd KVI-Integration
  python -m venv env
```

Activate the Environment

```bash
  # Windows
  env\Scripts\Activate

  # Unix / OS Machine
  source env/bin/activate
```

Run the App

```bash
py -m main
or
python main.py
```
